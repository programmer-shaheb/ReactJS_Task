## Getting Started

### Prerequisites

Install the dependencies:

- npm
  ```sh
  npm install
  ```

### Installation

- Clone the repo
  ```sh
  git clone https://github.com/programmer-shaheb/ReactJS_Task.git
  ```
- Install NPM packages
  ```sh
  npm install
  ```
